# Adivina el Número

Juego web donde el usuario intenta adivinar un número secreto generado aleatoriamente entre 1 y 100. La aplicación da pistas de "mayor" o "menor" en cada intento y lleva un contador, con un límite máximo de intentos.

## Tecnologías

- HTML5
- JavaScript (Vanilla, sin frameworks)

## Funcionalidades

- Generación de número aleatorio en cada partida
- Validación del intento del usuario
- Contador de intentos con límite máximo
- Botón para reiniciar el juego sin recargar la página

## Cómo ejecutarlo

1. Clona este repositorio o descarga los archivos
2. Abre `index.html` en tu navegador
